## 🏁 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
pip install -r requirements.txt

```

### 3. Run the Flask Server

```bash
python flask_app.py
```

the open your browser and go to:

```bash
http://localhost:5000
```
